发表于 2020-12-28 | 分类于 spring

依赖注入的模式和类型

模式区别

手动模式 - 配置或者编程的方式，提前安排注入规则
  • XML 资源配置元信息
  • Java 注解配置元信息
  • API 配置元信息
自动模式 - 实现方提供依赖自动关联的方式，按照內建的注入规则

  • Autowiring(自动绑定)

类型区别

自动模式只支持setter和构造器类型的注入，其中setter需要指定byName或byType

Aware接口回调注入

主要作用是注入spring内建的接口，比如获取以下常用的spring组件

依赖注入类型选择

  • 低依赖:构造器注入
  • 多依赖:Setter 方法注入 • 便利性:字段注入
  • 声明类:方法注入
