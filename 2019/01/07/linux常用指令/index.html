linux常用命令整理

发表于 2019-01-07 | 分类于 工具

文件查找：

find ~ -name “xx.txt”

查找~路径下的xx.txt文件

-iname 忽略大小写，支持通配符匹配

检索文件内容

grep -i ‘xxx’ log.log 检索单行

| 管道命令，把上一个命令的输出流传递给下一个命令的输入流。不支持输入流的命令会把数据抛弃

awk -F “,” ‘{print $1}’ log.log

以逗号作为分隔符，打印分割后第一个变量

清理冗余PATH

export PATH=$(echo $PATH | tr : “\n”| sort | uniq | tr “\n” :)
