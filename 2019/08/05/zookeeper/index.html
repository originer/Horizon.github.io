zookeeper领导者选举

发表于 2019-08-05 | 分类于 zookeeper

zookeeper的定义

zookeeper本质是一个分布式数据库；

增加写性能：两阶段发送

增加读性能：添加observer节点，只读不写

脑裂解决方法：过半机制 zookeeper集群节点个数最好是单数

zk数据一致性保证的是CP C 一致性 A 可用性 P 分区容错性

领导者选举过程

1.先投给自己

2.比较zxid(事务id)，如果一样就比较myid

3.投票给数据更新的节点

4.投票箱统计

5.获取票数最多的人当选
