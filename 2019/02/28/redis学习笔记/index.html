发表于 2019-02-28 | 分类于 redis

底层数据结构

简单字符串sds，类似StringBuilder

对比常规字符串的优势：

本身记录长度，获取长度的复杂度为O(1)

杜绝缓冲区溢出，即可以自动扩容。

使用空间预分配和惰性空间释放来优化内存分配，避免频繁分配内存带来的性能开销。

二进制安全，可以保存任意格式的二进制数据。
